# About

These are recreations of Cave Story entities in ModCS Lua. The only way you can use this code is with the ModCS lua api for Cave Story.

It is recommended to only copy the Npcs that you need. If you want to modify the Igor boss entity for example, only copy that Npc ID's code, not the entire file containing it.

These have been tested, but there still may be some parts of code that are inaccurate/behave incorrectly. Also, sometimes an Npc will use npc.xm_cs / npc.ym_cs, instead of the ModCS style values to ensure accuracy.
